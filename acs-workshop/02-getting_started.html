ACS Architecture

The StackRox Kubernetes Security Platform installs as a set of pods in your Kubernetes or OpenShift cluster and includes the following components:

  • Central: Gathers and displays information from other components (1 for multiple clusters).

  • Sensor: Collects and augments data from the Collector (1 for each cluster.)

  • Scanner: Scans images for vulnerabilities (1 for multiple clusters)

  • Collector: Collects and monitors container activities (1 on each node)

  • Admission controller: Interacts with Kubernetes API server and prevents creating workloads that don’t adhere to security policies (1 for each cluster)
